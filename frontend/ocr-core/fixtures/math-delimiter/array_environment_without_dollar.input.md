The orbital-element conversion table was recognized as a bare LaTeX array:

\begin{array}{ll}
x = r(\cos\Omega\cos u-\sin\Omega\sin u\cos i), & y = r(\sin\Omega\cos u+\cos\Omega\sin u\cos i),\\
z = r\sin u\sin i, & u = \omega + f.
\end{array}

Here \(f\) is the true anomaly measured from perihelion.
